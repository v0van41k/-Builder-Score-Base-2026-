# GitHub для Builder Score

GitHub — один из самых важных факторов.

## Что помогает:

- Возраст аккаунта
- Количество репозиториев
- Коммиты (особенно с кодом)
- Stars и watchers
- Вклад в популярные проекты

**Рекомендации**:
- Делай 3–5 коммитов в неделю
- Пиши README на английском
- Добавляй скриншоты и инструкции
- Создавай полезные инструменты
